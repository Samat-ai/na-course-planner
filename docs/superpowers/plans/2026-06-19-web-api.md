# NA Course Planner — Plan 4: Web API + Minimal Test UI

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expose the full pipeline (parse → audit → recommend → export) as a clean, stateless FastAPI JSON API, with a deliberately minimal throwaway UI to exercise it end-to-end.

**Architecture:** API-first. FastAPI wraps the pure engine (Plans 1–2) and ingestion (Plan 3); it holds no logic of its own beyond request/response mapping. **Stateless — the client carries the `StudentRecord`** and sends it with each request; the server stores nothing. A small program registry loads catalog YAML by code+year. Export produces JSON (re-importable) and PDF (printable). A single static HTML page drives the API for manual testing; the real frontend is a later, independent client.

**Tech Stack:** Python 3.13, FastAPI, uvicorn, python-multipart (uploads), fpdf2 (PDF, added in Plan 3), httpx (TestClient, dev), pytest.

## Global Constraints

- All Plan 1 constraints apply (`py -3`; Python ≥3.13; Pydantic v2; `src/` layout; TDD; commit per task).
- **Stateless — server stores nothing.** No database, no server-side session, no transcript written to disk or held beyond a request. The client sends the `StudentRecord` each call.
- **The API contains no business logic** — it calls `audit`, `recommend`, `parse_transcript_*`, `to_student_record`, and the export functions. Any logic bug belongs in the engine plans, not here.
- **Programs are loaded from `data/programs/` by `code` + `catalog_year`** via the registry; unknown program → HTTP 404.
- **Image-only PDF upload → HTTP 422** with a message telling the user to paste/enter manually (never a 500).
- **CORS:** allow all origins in v1 (it's a stateless public tool with no secrets); tighten when the real frontend has a known origin.

```python
from pathlib import Path

from na_planner.catalog_loader import load_program
from na_planner.models.catalog import Program

PROGRAMS_DIR = Path(__file__).parents[2] / "data" / "programs"


def list_programs(directory: Path = PROGRAMS_DIR) -> list[dict]:
    out: list[dict] = []
    for path in sorted(directory.glob("*.yaml")):
        p = load_program(path)
        out.append({"code": p.code, "name": p.name, "catalog_year": p.catalog_year})
    return out


def load_program_by(
    code: str, catalog_year: int, directory: Path = PROGRAMS_DIR
) -> Program:
    for path in sorted(directory.glob("*.yaml")):
        p = load_program(path)
        if p.code == code and p.catalog_year == catalog_year:
            return p
    raise KeyError(f"No program {code} for catalog year {catalog_year}")
```

```python
from fastapi import FastAPI, HTTPException
from fastapi.middleware.cors import CORSMiddleware

from na_planner.api.schemas import AuditRequest
from na_planner.audit import audit
from na_planner.models.audit import AuditResult
from na_planner.programs import list_programs, load_program_by


def create_app() -> FastAPI:
    app = FastAPI(title="NA Course Planner API", version="0.1.0")
    app.add_middleware(
        CORSMiddleware, allow_origins=["*"], allow_methods=["*"], allow_headers=["*"],
    )

    @app.get("/health")
    def health() -> dict:
        return {"status": "ok"}

    @app.get("/programs")
    def programs() -> list[dict]:
        return list_programs()

    @app.post("/audit", response_model=AuditResult)
    def audit_endpoint(req: AuditRequest) -> AuditResult:
        try:
            program = load_program_by(req.program_code, req.catalog_year)
        except KeyError as e:
            raise HTTPException(status_code=404, detail=str(e)) from e
        return audit(req.student, program)

    return app


app = create_app()
```

```python
from fpdf import FPDF

from na_planner.models.recommend import Recommendation


def plan_to_json(rec: Recommendation) -> bytes:
    return rec.model_dump_json(indent=2).encode("utf-8")


def _term_lines(pdf: FPDF, title: str, term) -> None:
    pdf.set_font("Helvetica", style="B", size=12)
    pdf.cell(0, 8, f"{title}: {term.label} ({term.total_credits:.0f} cr)",
             new_x="LMARGIN", new_y="NEXT")
    pdf.set_font("Helvetica", size=10)
    for c in term.courses:
        reasons = f" — {', '.join(c.reasons)}" if c.reasons else ""
        pdf.cell(0, 6, f"  - {c.code} ({c.credits:.0f} cr){reasons}",
                 new_x="LMARGIN", new_y="NEXT")
    for w in term.warnings:
        pdf.cell(0, 6, f"  ! {w}", new_x="LMARGIN", new_y="NEXT")


def plan_to_pdf(rec: Recommendation) -> bytes:
    pdf = FPDF()
    pdf.add_page()
    pdf.set_font("Helvetica", style="B", size=16)
    pdf.cell(0, 10, "NA Course Plan (tentative)", new_x="LMARGIN", new_y="NEXT")
    _term_lines(pdf, "Next term", rec.next_term)
    for t in rec.roadmap:
        _term_lines(pdf, "Then", t)
    pdf.set_font("Helvetica", size=10)
    pdf.cell(0, 8, f"Projected graduation: {rec.projected_graduation or 'TBD'}",
             new_x="LMARGIN", new_y="NEXT")
    pdf.cell(0, 6, f"Elective credits remaining: {rec.elective_credits_remaining:.0f}",
             new_x="LMARGIN", new_y="NEXT")
    pdf.cell(0, 6, "Advisory only — verify with your advisor and the registrar.",
             new_x="LMARGIN", new_y="NEXT")
    return bytes(pdf.output())
```

- **CORS is wide open** (`*`) — acceptable for a stateless tool with no secrets; tighten to the real frontend origin later.
